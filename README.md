# Ruhullah Najafi - Portfolio Website

A modern, tech-oriented portfolio website built with Next.js, featuring sleek animations and non-conventional design elements.

## About

Personal portfolio showcasing my journey as a SAP Consultant, with educational background from TUHH Hamburg and UC Berkeley, and a passion for Mathematics and Science.

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Font**: System UI fonts

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

```bash
npm run build
npm start
```
